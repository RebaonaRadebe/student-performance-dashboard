# 📊 Student Performance Analytics Dashboard (Excel Project)

## 📌 Overview
This project is an Excel-based data analytics dashboard that analyses student performance across different modules. It was built to practice real-world data analytics skills including data summarisation, pivot tables, visualisation, and dashboard design.

The dashboard helps identify overall performance trends, pass/fail distribution, and module-level performance insights.

---

## 🎯 Objectives
- Analyse student academic performance data
- Track pass vs fail distribution
- Compare performance across modules
- Calculate key performance indicators (KPIs)
- Build an interactive dashboard using Excel

---

## 🛠 Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (interactive filters)
- Data aggregation (average marks, counts)
- Dashboard design principles

---

## 📊 Dashboard Features
The dashboard includes:

- **KPI Metrics**
  - Total number of students
  - Number of students passed
  - Overall pass rate

- **Visual Analysis**
  - Pie chart showing Pass vs Fail distribution
  - Column chart showing average marks per module

- **Interactivity**
  - Slicers to filter by module
  - Slicers to filter by student status

---

## 📈 Key Insights
- Overall pass rate: **90%**
- Total students analysed: **30**
- Students who failed: **3**
- Highest performing module: **Internet Programming (74.36 average mark)**
- Lowest performing module: **Database Systems (70.25 average mark)**
